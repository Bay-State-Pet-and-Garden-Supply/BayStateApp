## üîß Admin Portal

### General Modal Actions
Applies to most management modals (Products, Services, Brands, Orders).

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl + S` / `Cmd + S` | **Save Changes** | Persist changes in the current modal. |
| `Esc` | **Close Modal** | Dismiss the current modal without saving. |

### ü§ñ Workflow Builder
Specific to the Scraper Workflow Builder interface.

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Cmd + Z` (Mac)<br>`Ctrl + Z` (Win) | **Undo** | Revert the last change to the workflow nodes. |
| `Cmd + Shift + Z` (Mac)<br>`Ctrl + Shift + Z` (Win) | **Redo** | Reapply a previously undone change. |

### üìù Scraper Editor
Specific to the YAML Configuration Editor.

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Cmd + S` / `Ctrl + S` | **Save Config** | Save the current scraper configuration. |
| `Shift + Alt + F` | **Format Code** | Auto-format the YAML configuration. |
| `Cmd + F` / `Ctrl + F` | **Find** | Open the search widget within the editor. |
| `Cmd + K` then `Cmd + 0` | **Fold All** | Collapse all regions in the editor. |
